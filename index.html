Keamanan Password

Disusun Oleh Hermina Jakaria. Pada 26 Januari 2022

Password atau passphrase berasal dari Bahasa inggris yang berarti kata sandi, Password adalah sekumpulan karakter atau string yang digunakan oleh pengguna jaringan atau sebuah system oprasi.
Kegunannya untuk memverifikasi identitas dirinya kepada system keamanan yang dimiliki oleh jaringan atau system tersebut, kata sandi juga dapat digunakan sebagai tanda pengenal.

Resiko Penggunaan Password

Resiko dari penggunaan password dimana password kita dalam sebuah system di ketahui oleh orang yang tidak bertanggung jawab, faktor dari resiko penggunaan password diakibatakan oleh antara lain sperti:

  • Kata sandi/password yang lemah
  • Software/kelemahan pada system oprasi
  • Human error
